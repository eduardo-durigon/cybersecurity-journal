## 💡 Key Concepts

### Network Addressing and Communication

Module 6 introduced the foundations of TCP/IP communication and how devices identify and communicate with one another across networks.

Topics included:

* IPv4 and IPv6 addressing
* Public and private IP addresses
* Subnet masks and prefix notation
* Default gateways
* Routing fundamentals
* Network Address Translation (NAT)

Learning how IP addressing works helped me better understand how devices locate one another and communicate both locally and across the internet.

### DHCP and DNS Services

I also studied two critical services that support modern networks.

Topics included:

* Dynamic Host Configuration Protocol (DHCP)
* DHCP leasing process
* Domain Name System (DNS)
* DNS hierarchy
* Resource records
* Name resolution

Understanding DHCP and DNS made it much easier to visualize how devices automatically obtain network settings and how websites can be accessed using domain names instead of numerical IP addresses.

### TCP and UDP Communication

Another major topic was transport-layer communication.

Topics included:

* TCP three-way handshake
* Reliable communication
* Error checking and retransmission
* UDP communication
* Connection-oriented vs connectionless protocols
* Protocol reliability and performance trade-offs

This section helped me understand why different applications choose either TCP or UDP depending on their requirements.

### Network Security and Segmentation

The module also introduced concepts commonly used in enterprise environments.

Topics included:

* VLANs
* VPNs
* Secure remote access
* Network segmentation
* Encrypted communications

These concepts demonstrated how organizations improve both security and network management by separating traffic and protecting data in transit.

## 🚀 Key Takeaways

* IP addressing forms the foundation of network communication.
* DHCP and DNS are essential services that support everyday network operations.
* TCP prioritizes reliability while UDP prioritizes speed.
* VLANs improve network segmentation and security.
* VPNs provide secure communication across untrusted networks.
* Practical troubleshooting skills are just as important as theoretical knowledge.
* Terminal-based networking tools provide valuable insight into system communication.
* Hands-on labs significantly improve retention and understanding.
* Hardware, operating systems, networking, and security are all interconnected.
