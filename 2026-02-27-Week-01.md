## 2️⃣ Key Concepts Learned

- **MAC vs IP Addressing:**  
  A MAC address is a physical, hardware-based identifier used inside a LAN. An IP address is a logical address used to route traffic across networks. Understanding the distinction is essential when troubleshooting or analyzing network traffic.

- **Wireless Security Standards:**  
  WEP is deprecated due to weak encryption. WPA2 introduced AES encryption, and WPA3 is currently the most secure standard. Securing wireless access points is a primary layer of defense.

- **Latency (Ping):**  
  Latency measures delay in milliseconds and affects real-time applications like gaming and VoIP. High latency can indicate congestion, distance issues, or network misconfiguration.

- **Virtualization & Hypervisors:**  
  Type 1 hypervisors run directly on hardware and are common in enterprise environments. Type 2 hypervisors run inside a host OS and are common in labs and development environments.
