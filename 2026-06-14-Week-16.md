## Hands-On Lab Work

This week I completed a practical networking lab using macOS command-line tools.

Tools used included:

* dig
* nslookup
* curl
* openssl
* traceroute
* netstat
* lsof
* netcat
* ifconfig
* route
* sntp

During the lab I:

* Tested DNS resolution
* Compared HTTP and HTTPS communication
* Inspected TLS certificates
* Tested common service ports
* Queried DNS records
* Verified time synchronization
* Traced network paths
* Reviewed active network connections

The lab reinforced how networking protocols, services, ports, and security controls interact in real-world environments.

---

## Key Takeaways

One of the biggest lessons from this week was seeing how much information can be gathered and verified directly from the command line.

Tools such as dig, curl, openssl, traceroute, netstat, and lsof provide visibility into network behavior, connectivity, encryption, routing, and active communications. These are foundational skills not only for IT support but also for future cybersecurity work.

The virtualization concepts from Module 8 were also particularly interesting because they demonstrate how modern organizations efficiently deploy systems, isolate workloads, and manage infrastructure through virtual machines and cloud platforms.
