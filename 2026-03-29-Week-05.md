## 2️⃣ Key Concepts Learned  

### 🗄️ Database Fundamentals  

- Databases store and organize data efficiently  
- Data is structured into:
  - **Tables, records, and fields**  
- Managed using a **DBMS (Database Management System)**  

---

### 🔑 Keys & Relationships  

- **Primary Key** → unique identifier  
- **Foreign Key** → links tables together  

- Relationships:
  - One-to-One  
  - One-to-Many  

These are essential for maintaining data integrity.

---

### 📊 Types of Databases  

- **Relational (SQL)**:
  - Structured tables  
  - Strong relationships  
- **Non-relational (NoSQL)**:
  - Flexible structure  
  - Used in large-scale applications  

---

### 🔍 Queries & Data Handling  

- Queries retrieve and manipulate data  
- SQL is commonly used  
- Data can be:
  - Structured  
  - Unstructured  
  - Semi-structured  

---

### 💻 Coding Fundamentals  

- Coding is writing instructions for computers  
- Computers operate using **binary (1s and 0s)**  
- Programming languages translate logic into machine-readable instructions  

---

### 🧠 Syntax vs Semantics  

- **Syntax** → structure  
- **Semantics** → meaning  

Both must be correct for programs to work properly.

---

### 🔁 Control Flow  

- **Sequential** → step-by-step  
- **Conditional** → decisions (if/else)  
- **Loops** → repetition  
- **Functions** → reusable code  

---

### 📊 Variables & Data Types  

- Variables store data  
- Common types:
  - Integer  
  - Float  
  - String  
  - Boolean  
  - Arrays/Lists  

---

### 🧱 Programming Concepts  

- Functions improve code reuse  
- **OOP (Object-Oriented Programming)**:
  - Classes and objects  
  - Properties and methods  

---

### 🗺️ Planning & Logic  

- Flowcharts → visualize logic  
- Pseudocode → write logic in plain language  
- Algorithms → step-by-step problem solving  

---

### 🐧 Linux Fundamentals (Hands-On Lab)  

Commands practiced:

- `pwd` → current directory  
- `ls` → list files  
- `cd` → navigate  
- `mkdir` → create folders  
- `touch` → create files  
- `cat` → read files  
- `chmod` → change permissions  

---

### 🔐 Cybersecurity Context  

- Investigated simulated suspicious activity  
- Reviewed log files  
- Inspected a shell script  
- Secured files using permissions  

This showed how Linux is widely used in cybersecurity environments.
