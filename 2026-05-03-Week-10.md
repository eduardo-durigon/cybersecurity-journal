### 🛠️ Lab Work – Python Lab 02: Password Strength Checker

#### Overview

This lab focused on building a simple password strength checker using Python.

The script:
- Prompts the user to enter a password
- Evaluates it against basic security rules
- Assigns a strength rating (Weak / Medium / Strong)

#### What the Script Checks

- Password length  
- Presence of uppercase letters  
- Presence of lowercase letters  
- Presence of numbers  

#### Skills Practiced

- Variables and user input (`input()`)
- Conditional logic
- Loops (`for`)
- String analysis methods
- Lists and `.append()`
- Running scripts via terminal
