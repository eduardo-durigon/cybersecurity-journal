## 💡 Key Concepts

### Power Supplies and Cooling

One of the first topics covered was how power is delivered throughout a computer system.

Topics included:

* PSU wattage ratings
* Power rails and voltage distribution
* 80 Plus efficiency certifications
* Modular and non-modular power supplies
* Redundant power supplies
* Cooling systems and airflow management

I learned how choosing the correct power supply is critical for system stability and how efficient cooling helps protect hardware from overheating and performance issues.

### Storage Technologies

A major portion of the module focused on storage devices and their performance characteristics.

Topics included:

* Hard Disk Drives (HDDs)
* Solid State Drives (SSDs)
* NVMe storage
* SATA interfaces
* M.2 form factors
* Optical media
* Flash storage and SD cards

One of the most interesting lessons was seeing the performance differences between traditional hard drives and modern NVMe SSDs. It became clear why SSD technology has become the standard for modern systems.

### RAID and Data Redundancy

I also learned about RAID configurations and how organizations improve storage reliability.

Topics included:

* RAID 0
* RAID 1
* RAID 5
* RAID 6
* RAID 10

Understanding the balance between performance, redundancy, and fault tolerance gave me a better appreciation for how businesses protect critical data.

### Memory and CPU Technologies

The second half of the module focused on memory and processor technologies.

Topics included:

* DDR memory generations
* DIMM and SODIMM modules
* Memory channels
* ECC memory
* x86 and x64 processors
* ARM architecture
* CPU sockets and installation
* Virtualization technologies

I found it particularly interesting to learn how modern CPUs use multiple cores, cache memory, and virtualization features to improve performance and support modern computing environments.

## 🚀 Key Takeaways

* Reliable power delivery is essential for system stability and performance.
* Cooling systems play a critical role in preventing hardware failures and thermal throttling.
* SSDs and NVMe drives provide major performance advantages over traditional HDDs.
* RAID configurations can improve redundancy, performance, or both depending on the implementation.
* Modern CPUs rely on multicore processing, cache memory, and virtualization technologies.
* Hardware compatibility is extremely important when upgrading or building computer systems.
* Understanding system hardware provides a strong foundation for future IT and cybersecurity studies.
